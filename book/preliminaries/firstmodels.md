# Your very first models

Before diving into the topic, we can gain some machine learning intuition through a few interactive k-Nearest Neighbors models. The only thing you need to understand at this point is that we would like to find underlying patterns in some regression data by fitting it: by looking at more neighbors we end up with more rigid and smoother models; by looking at at small neighborhood we end up with very flexible models.
